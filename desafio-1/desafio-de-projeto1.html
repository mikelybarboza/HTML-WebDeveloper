Introdução ao HTML

(Voltar)

HTML significa Hyper Text Markup Language. Ele é usado no front-end e fornece a estrutura da página da Web que você pode estilizar usando CSS e tornar interativa usando JavaScript

Estrutura Básica do HTML

(Voltar)

< !DOCTYPE html>
< html>
< head>
< title> < /title>
< body>
< h1> < /h1>
< p> < /p>
< /body>
< /html>

Falando Sobre tags

(Voltar)

  • < !DOCTYPE html> esta declaração define que este documento é um documento HTML5
  • O < html>elemento é o lemento raiz de uma página
  • < title> este elemento especifica um titulo da página HTML (que é mostrado na brra do título do navegador ou na guia de página)
  • < body> este elemento define o corpo do documeto é um recipiente para todos os conteudos visíveis, como cabeçalhos, parágrafos, imagens, hiperinks, tabelas, listas, etc...
  • O < h1> elemento define um tituo grande
  • O < p> elemento define um paragrafo

Atributos básicos das tags

(Voltar)

  • Todos os elementos HTM podem ter atributos
  • Os atributos fornecem informações adicionais sobre os elemntos
  • Os atributos são sempre especificados na tag de inicio
  • Os atributos gerlmente vêm em pares de nome/valor como: name="value"
